# Prompt Ops Copilot

## What This Is
A tool that transforms amateur development issues into professional, agent-ready prompts optimized for AI coding tools (Cursor, Lovable, Replit, Codex). Supports Hebrew input with automatic English translation.

## Core Value
Helps developers write better prompts for AI coding assistants by providing structure, best practices, and agent-specific formatting.

## Tech Stack
- **Frontend:** Next.js 15 (App Router), React 19, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Next.js API Routes, Supabase (PostgreSQL + Auth)
- **AI:** OpenAI, Anthropic, Groq provider abstraction
- **Monorepo:** pnpm + Turborepo

## Current State
Extended MVP with core features implemented:
- Composer (Quick Convert) with RTL support
- Prompt Library with search/filter
- Project Management with context packs
- Custom Agents with CRUD
- 11 built-in target agents
